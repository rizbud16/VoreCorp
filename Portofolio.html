Tentang Kami

Vore Corporation adalah sebuah perusahaan virtual yang bergerak dibidang pengembangan Blogger & Web Design. Selain itu kami juga bergerak dalam bidang pengembangan aplikasi desktop berbasis sistem operasi Windows yang memiliki lisensi maupun open source. Pengembangan web yang kami kuasai untuk saat ini yaitu Front-end termasuk jQuery dan javascript framework lainnya serta Bootstrap dan css framework lainnya, untuk Back-end yaitu PHP dan Codeigniter untuk php frameworknya.

Dalam pengembangan aplikasi desktop berbasis sistem operasi Windows kami menggunakan bahasa pemrograman Visual Basic dengan Visual Studio sebagai alat membuatnya. Selain bergerak dibidang teknologi, kami juga menjual beberapa produk souvenir seperti Cangkir, Gantungan Kunci, Pin, dan lain-lainnya.
